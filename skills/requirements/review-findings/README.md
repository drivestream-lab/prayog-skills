# review-findings

Interactive workshop for validation / audit reports. **Collect decisions only** — does not edit the PRD. Writes `Resolution-*.md` next to the report in the consumer repo (e.g. `prd/reports/`).

**Original author:** rushikeshpol02 (ai-skills). **Maintainer:** drivestream-lab.

## Invoke

```
/review-findings prd/reports/Validation-Report-INIT-PRAYOG-001.md
```

Run after `validate-requirements`. Pair with manual PRD / stub updates, then re-run validation in incremental mode.

See repo root README for install command.
